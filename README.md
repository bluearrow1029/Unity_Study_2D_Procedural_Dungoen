UnityEngine.Tilemaps가 이용됨.
PaintSingleTile 메소드에서 tilemap의 SetTile 메소드를 이용함.
매개변수로 받은 position을 월드 기준의 좌표 tilePosition으로 설정함.
특정 tilemap에서 지정한 tile을 월드 기준 좌표인 tilePosition 좌표에 배치함.
